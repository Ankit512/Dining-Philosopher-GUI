========================
BUILD OUTPUT DESCRIPTION
========================

To run the project from the command line, go to the dist folder and
type the following:

java -jar "DiningPhilosoperWithGUI.jar" 

To distribute this project, zip up the dist folder (including the lib folder)
and distribute the ZIP file.

To run this project on NetBeans IDE follow these steps:
1. Open NetBeans
2. Go-to File -> Open Project -> Select the paticular project(filename: OS Project_Ankit Kumar_RA1711003010086) -> Open Project.
3. Expand "DiningPhilosopherWithGUI" class.
DiningPhilosopherWithGUI -> Source Packages -> Philosoper
4. Click on the run icon to execute the program.
